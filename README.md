# Student Marks Manager

A simple web application built using **HTML, CSS, and JavaScript** that allows users to add, view, and delete student marks with automatic grade calculation.

---

## 🚀 Features

- Add student name and marks  
- Automatic grade calculation  
- Display data in table format  
- Delete student records  
- Data stored using `localStorage`  
- Input validation (0–100 marks only)

---

## 🧠 Grade Logic

| Marks Range | Grade |
|------------|-------|
| 90 – 100 | A |
| 75 – 89 | B |
| 60 – 74 | C |
| 40 – 59 | D |
| Below 40 | F |

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Browser localStorage
